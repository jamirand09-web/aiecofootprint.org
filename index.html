Methodology Notes

This calculator uses averaged peer-reviewed baselines and regional grid profiles. Actual prompt impact depends on model architecture, server utilization, and local data center practices.

  • Values are standardized per token and use averaged regional multipliers rather than live telemetry.
  • Water estimates reflect blended direct and upstream effects, not a single facility’s cooling system.
  • These results are intended as an order-of-magnitude estimate and conversation starter, not a precise carbon ledger.

If you have more granular infrastructure data or model-specific benchmarks, open an issue on GitHub to help refine the model.
